## AI Agent Basic Workflow
<b>Run Repo:</b>

Tải Ollama 

https://ollama.com/download/windows

Mở terminal: ollama pull gemma3:4b

```
cd webapp
cd frontend
npm install
npm run dev
cd ../backend
pip install -r requirements.txt
python main.py 
```
